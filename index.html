Цели

Моя цель — стать опытным веб-разработчиком полного цикла с прочной базой знаний в области как фронтенд-, так и бэкенд-технологий, а затем постепенно перейти в сферу машинного обучения. Чтобы добиться этого, я составил четкую и структурированную 6-месячную дорожную карту:

Месяцы 1–2: Я начинаю с изучения основ — HTML, CSS, Git и GitHub — и создаю сайт-портфолио с формой обратной связи. Затем я погружаюсь в основы JavaScript и манипуляции DOM, создавая интерактивную веб-страницу.

Месяцы 2–3: Я углубляю свое понимание JavaScript и изучаю React, применяя его на сайте To-Do или Movie. Затем я работаю с React Router, Tailwind CSS и внешними API, создавая приложение React, которое извлекает данные из внешних сервисов.

Месяцы 3–4: Я переключаю фокус на разработку бэкенда — изучаю Python, FastAPI и SQL — и создаю бэкенд-проект с базой данных. Затем я перехожу к аутентификации JWT, SQLAlchemy и API фронтенда, создавая блог или менеджер задач. После этого я интегрирую Axios, аутентификацию пользователей и базовый Docker, в результате чего получается полнофункциональное приложение.

Месяцы 4–5: Я изучаю, как развертывать frontend и backend приложения с помощью Vercel и Render. Затем я начинаю работать с Python для анализа данных, используя pandas и matplotlib для создания диаграмм и анализа данных.

Месяцы 5–6: Я погружаюсь в машинное обучение, начиная с scikit-learn и алгоритмов машинного обучения, и создаю модель машинного обучения для прогнозирования. Затем я интегрирую эту модель в FastAPI, чтобы создать API на основе машинного обучения, и, наконец, объединяю ее с интерфейсом, чтобы создать полноценное веб-приложение ИИ для анализа текста.
